Ollama 설치

Ollama 다운로드 →

Step 2

Git 설치

소스코드를 내려받기 위해 Git이 필요합니다.

Git 다운로드 →
💡 설치 중 옵션은 전부 기본값(Next)으로 진행하면 됩니다.
Step 3

Gemma AI 설치

터미널(명령 프롬프트)을 열고 아래 명령어를 복사해서 붙여넣으세요.

powershell -ExecutionPolicy Bypass -c "irm https://raw.githubusercontent.com/johnsnow-nam/gemma4/master/install.ps1 | iex"
curl -sSL https://raw.githubusercontent.com/johnsnow-nam/gemma4/master/install.sh | bash
curl -sSL https://raw.githubusercontent.com/johnsnow-nam/gemma4/master/install.sh | bash
💡 설치에 5~10분 정도 걸려요. AI 모델을 다운로드하기 때문이에요.
Step 4

실행하기

설치가 끝나면 바탕화면에 Gemma AI 바로가기가 생겨요. 더블클릭하면 바로 시작됩니다!
