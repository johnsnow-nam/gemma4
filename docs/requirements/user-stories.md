# 사용자 스토리

## 페르소나
**Yonghee** — 임베디드 개발자. ESP32 OCPP 충전기 펌웨어 개발 중.
로컬 AI로 코드 작업을 돕는 도구가 필요함. 이동 중에도 원격으로 서버 작업 가능 원함.

---

## gemma-cli

- 개발자로서, 터미널에서 AI와 대화하며 C 코드를 분석하고 싶다.
- 개발자로서, `@uart.c`처럼 파일을 첨부해 AI에게 코드 리뷰를 받고 싶다.
- 개발자로서, AI가 제안한 코드를 바로 저장/실행할 수 있어야 한다.
- 개발자로서, `/watch` 로 파일 저장 시 자동으로 AI 분석을 받고 싶다.
- 개발자로서, git 변경사항을 AI가 자동으로 커밋 메시지를 만들어줬으면 한다.

## gemma-desktop-mcp

- Claude Desktop 사용자로서, Claude에서 직접 로컬 파일을 읽고 Gemma4에 분석을 요청하고 싶다.
- 개발자로서, Claude Desktop에서 셸 명령어를 실행하고 결과를 AI가 해석해줬으면 한다.
- 개발자로서, 회로도 이미지를 Claude Desktop에 붙여넣고 Gemma4가 분석해줬으면 한다.

## telegram-agent

- 이동 중인 개발자로서, 스마트폰 텔레그램으로 서버에서 빌드를 실행하고 싶다.
- 개발자로서, 빌드 실패 시 AI가 에러를 분석하고 수정 방법을 텔레그램으로 알려줬으면 한다.
- 개발자로서, 텔레그램으로 회로도 사진을 보내면 AI가 분석해줬으면 한다.
- 개발자로서, 외출 중에도 "uart.c 고쳐줘"라고 보내면 AI가 코드를 수정하고 커밋해줬으면 한다.
