## 4. gemma-cli 테스트 (STEP 1~11)

```bash
cd ~/projects/gemma4/gemma-cli
python3 gemma-cli.py

```

### STEP 3 — 파일 읽기 (@)

```

나 > @gemma-cli.py 이 파일의 구조를 설명해줘

# 다중 파일
```

**확인 포인트:**
- 파일 메타 정보(크기, 줄 수, 수정일) 출력
- 코드블록 언어 하이라이팅
